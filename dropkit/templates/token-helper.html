dropkit API token helper

This page helps you create a DigitalOcean API token with the scopes dropkit needs.

Fast scope selection

  1. Drag dropkit scopes to your bookmarks bar.
  2. Open the DigitalOcean API token creation page.
  3. Click the dropkit scopes bookmark you just created.
  4. It fills in the token name dropkit-cli and selects the required custom scopes.
  5. Review the selected scopes. There should be 23 selected scopes, then click Generate Token.
  6. Copy the token and paste it into dropkit init.

If your browser hides the bookmarks bar, enable it first, then drag the blue link above into it.
